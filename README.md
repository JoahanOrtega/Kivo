# Kivo

Kivo — tu dinero, en movimiento.

Aplicación mobile-first y local-first para gestión de finanzas personales.

## Stack principal

- React Native + Expo + TypeScript
- Rust + Axum
- SQLite local
- PostgreSQL remoto

## Objetivo

Permitir registrar ingresos y egresos de forma rápida desde el celular, incluso sin conexión, y sincronizar la información posteriormente con un backend central.
